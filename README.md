# GenericRealmBot
A generic version of the Bappo Realm Bot with more limited features. Meant for Minecraft: Bedrock Edition Realms.

Environmental vars: MAIN_TOKEN, OPENXBL_KEY, XAPI_KEY, GLOT_KEY, CONFIG_URL
